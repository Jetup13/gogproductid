# How-to Use Tool

1. Vist tool [website](https://jetup13.github.io/gogproductid/gog.html)
2. Search game or pack field
3. Select ES-DE or Daijishou button (Select optional download cover art option)
4. Genera zip
5. Unzip files into folder
